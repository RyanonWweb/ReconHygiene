# Security Policy

## Reporting a Vulnerability

If you discover a security issue related to this project or the deployed site (reconhygiene.com), please report it responsibly.

Do not publicly disclose the issue until it has been reviewed.

### Contact

Email: Reconhygieneops@protonmail.com

We aim to respond within **5 business days**.

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested remediation (if known)

## Scope

This policy applies to:

- reconhygiene.com
- All public repositories under this project
- Netlify deployment configuration

Thank you for practicing responsible disclosure.
